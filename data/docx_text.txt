1. 实验设计目标与研究假设
本实验设计面向受限水域多船会遇下的自主船局部避碰问题。实验目标不是验证某一成熟控制理论本身是否有效，而是验证成熟控制理论在真实航行约束下的系统化融合能力：当目标船具有真实船舶要素、多源规则同时生效、浅水/船岸/船间效应影响操纵、AIS 存在不确定性且控制需要实时执行时，所提出方法是否仍能生成安全、合规、物理可执行且具有泛化能力的避碰动作。
本文建议将方法命名为 TA-MRC-PE-CC-Tube-MPC，即 Target-Vessel-Aware, Multi-Rule-Constrained, Physics-Enhanced Chance-Constrained Tube-MPC。其核心不是重新发明 MPC、tube-MPC 或 CBF，而是将目标船全要素、多源规则、物理增强扰动、概率安全边界、鲁棒 tube 边界和运行时安全保障统一到一个可复现实验框架中。

4. 本船与目标船建模
4.1 本船 3-DOF/MMG 模型
本船采用 3-DOF/MMG 操纵模型作为主实验动力学基础。状态、速度和控制输入定义为：
η_i = [x_i, y_i, ψ_i]^T,     ν_i = [u_i, v_i, r_i]^T,     τ_i = [δ_i, n_i]^T
其中 x_i,y_i 为局部坐标，ψ_i 为艏向，u_i,v_i,r_i 分别为纵荡速度、横荡速度和艏摇角速度，δ_i 为舵角，n_i 为推进指令或螺旋桨转速。控制约束包括最大舵角、舵速、推进变化率、最小回转半径、最大艏摇角速度、航道边界和船岸安全距离。
4.2 目标船全要素建模
每艘目标船不再被建模为点障碍物，而被定义为具有尺度、速度、船型、航行状态和预测不确定性的动态船舶对象：

5. 会遇风险计算与边界条件处理
风险预测层首先计算两船相对位置和相对速度，并对 TCPA/DCPA 的边界情况进行严格处理，以避免目标船低速、锚泊或两船相对速度接近零时出现数值不稳定。
p_rel^{ij}=p_j-p_i,     v_rel^{ij}=v_j-v_i
TCPA_raw^{ij}= - (p_rel^{ij})^T v_rel^{ij} / (||v_rel^{ij}||^2 + ε_v)
TCPA^{ij}=clip(TCPA_raw^{ij}, 0, T_H)
DCPA^{ij}=||p_rel^{ij}+TCPA^{ij} v_rel^{ij}||

在分母加入 ε_v，并优先使用当前距离与安全域判断风险。

7. 目标船要素感知动态安全域
动态安全域采用本船-目标船耦合形式，基础安全域由两船尺度决定，膨胀项由速度、规则、目标船不确定性、水域环境、船岸效应和船间效应共同决定。所有膨胀项采用加法形式，避免危险因素增加时安全域反而减小的歧义。
d_safe,eff^{ij}=d_base^{ij}+Δd_speed^{ij}+Δd_size^{ij}+Δd_rule^{ij}+Δd_unc^{ij}+Δd_env^{ij}+Δd_bank^i+Δd_ship^{ij}+Δd_waterway^i

含义

保证目标船不是点障碍物。

7.1 量纲一致的安全域膨胀示例
Δd_speed^{ij}=α_v (L_i+L_j) (U_closing^{ij}/U_ref)
Δd_ship^{ij}=α_s (L_i+L_j) (U_rel^{ij}/U_ref) (d_ref/max(d_ij,d_min))
d_bank,safe^i = L_i [β_0 + β_1(B_i/L_i) + β_2(U_i/U_ref)^2 + β_3 I_shallow]
上述形式中 α、β 均为无量纲系数，U_ref 和 d_ref 为参考速度与参考距离，最终结果保持长度量纲。正式实验中必须通过参数敏感性证明结论不依赖单一人工参数。

9. 不确定性建模与机会约束确定化
机会约束不直接基于目标船协方差 Σ_j，而基于本船与目标船的相对位置协方差，保证概率安全边界的数学定义更加严谨。
p_rel^{ij}(t+k)=p_j(t+k)-p_i(t+k)
p_rel^{ij}(t+k) ~ N( μ_rel^{ij}(t+k), Σ_rel^{ij}(t+k) )
Σ_rel^{ij}(t+k)=Σ_j(t+k)+Σ_i(t+k)
若本船定位误差远小于 AIS 目标船预测误差，可令 Σ_i≈0；若使用实际传感器融合结果，则必须保留 Σ_i。
9.1 概率安全约束
P( d_ij(t+k) ≥ d_safe,eff^{ij}(t+k) ) ≥ 1 - ε
9.2 工程保守确定化
|| μ_rel^{ij}(t+k) || ≥ d_safe,eff^{ij}(t+k) + κ_ε sqrt( λ_max(Σ_rel^{ij}(t+k)) )
二维位置不确定性建议采用卡方置信椭圆：
κ_ε = sqrt( χ²_{2,1-ε} )

10. TA-MRC-PE-CC-Tube-MPC 控制问题
名义 MPC 生成满足规则、动力学和安全约束的控制序列；tube-MPC 使用扰动边界对状态安全集进行收缩，保证在 AIS、环境、船岸和船间扰动下仍具有鲁棒安全裕度。
min_{u_{0:N-1}} Σ_{k=0}^{N-1} [ ||x_k-x_ref,k||_Q² + ||u_k||_R² + ||Δu_k||_S² + J_rule,k + J_bank,k + J_ship,k ] + ||x_N-x_ref,N||_P²
约束包括：
x_{k+1}=f_MMG(x_k,u_k,w_k),     u_k ∈ U,     x_k ∈ X
δ_min ≤ δ_k ≤ δ_max,     |δ_k-δ_{k-1}|/Δt ≤ δdot_max
r_min ≤ r_k ≤ r_max,     R_turn,k ≥ R_min
d_bank,k ≥ d_bank,safe,k,     UKC_k ≥ UKC_min
|| μ_rel^{ij}(t+k) || ≥ d_safe,eff^{ij}(t+k) + κ_ε sqrt( λ_max(Σ_rel^{ij}(t+k)) ) + ρ_tube,k

ρ_tube = ρ_0 + ρ_AIS + ρ_current + ρ_wind + ρ_shallow + ρ_bank + ρ_ship + ρ_maneuver

11. CBF-QP 运行时安全过滤与 fallback
CBF-QP 不作为主要创新理论，而作为运行时安全过滤层。其作用是在 MPC 名义控制可能导致安全集边界被突破时，对控制输入进行最小修正。fallback 用于 MPC 不可行、CBF 过度介入、极端近距风险或传感器状态不可用时的保底策略。
u_safe = argmin_u ||u-u_MPC||²
s.t.  h_m(x) ≥ 0,     hdot_m(x,u) + α h_m(x) ≥ 0,     u ∈ U
安全函数
含义
h_ship^{ij}=d_ij-d_safe,eff^{ij}
本船-目标船安全域。
h_bank=d_bank-d_bank,safe
船岸安全距离。
h_UKC=UKC-UKC_min
富余水深。
h_lane
航道、TSS、分隔带和禁航区边界。

14.3 统计检验方案
正式实验每类核心场景建议至少 100 episodes、5 个随机种子；极端压力测试可增加至 200 episodes、10 个随机种子。结果报告应包含均值、标准差、95% 置信区间、median/IQR、worst-5% 性能和失败模式分布。
统计问题
建议方法
同一场景下方法两两比较
paired t-test 或 Wilcoxon signed-rank test。
多方法多指标比较
Holm-Bonferroni 或 Benjamini-Hochberg 校正。
效果大小
Cohen’s d 或 Cliff’s delta。
非独立扰动增强 episode
按真实 AIS 片段聚类 bootstrap 或分层统计。
多因素影响分析
mixed-effects model: Metric ~ Method + Scenario + Waterway + ShipType + Disturbance + (1|Seed)。
极端风险
worst-5% CPA、P95 runtime、max CBF intervention、fallback recovery。
