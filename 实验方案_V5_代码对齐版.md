## 1. Research Motivation

### 1.1 受限水域 MASS 多船避碰的核心挑战

自主船（MASS）在受限水域（港口、TSS 航道、狭窄水道、桥区）面临的多船避碰问题
无法由单一控制理论解决。真实航行中存在以下相互耦合的约束：

1. **目标船非点障碍物**: 真实目标船具有尺度、速度、船型、吃水、AIS 不确定性等要素，
   将目标船简化为点障碍物会导致安全域侵入。
2. **多源规则同时生效**: COLREGs、Inland Rules、TSS、VTS/VMRS、本地规则可能同时存在
   且相互冲突，需要优先级消解机制。
3. **物理效应影响操纵**: 浅水效应、船岸效应、船间效应、风流扰动会改变船舶操纵特性，
   受限水域下不可忽略。
4. **感知不确定性**: AIS 延迟/丢包/噪声使目标船状态预测存在概率不确定性。
5. **实时性约束**: 控制指令必须在秒级内计算完成，且需具备 infeasibility 恢复能力。

### 1.2 为什么现有方法不够

- **VO/DWA (B1/B2)**: 规则反应式，无法进行预测性优化，在受限水域多约束下难以找到可行解。
- **仅 COLREGs MPC (B3)**: 缺乏对物理效应和概率不确定性的建模，受限水域下鲁棒性不足。
- **单层安全机制**: 无 CBF 运行时过滤或无 fallback 恢复，极端场景下缺乏保护。

### 1.3 分层框架的必要性

本方案提出 TA-MRC-PE-CC-Tube-MPC 框架，将以下模块分层集成：

```
Layer 5: Fallback Supervisor (6-level recovery)
Layer 4: CBF-QP Safety Filter (runtime barrier-function correction)
Layer 3: Chance-Constrained Tube-MPC (optimization + robustness)
Layer 2: Physics Enhancement (shallow/bank/ship/wind/current)
Layer 1: Multi-Rule + Dynamic Ship Domain (P0-P5, COLREGs, VTS, ENC)
```

每一层提供独立的贡献，可通过 ablation 实验归因。

## 3. Method Overview

### 3.1 本船动力学模型

**方案真实实现**: 3-DOF 简化 MMG 操纵模型。

- **简化版**: 经验标度系数 (0.5ρL³, 0.5ρL⁴)，所有方法共享相同动力学基础。
- **标准版 (KVLCC2/KCS/KVLCC1)**: Yasukawa & Yoshimura (2015) 标准 MMG 系数，
  使用 U_design 标定的量纲化转换。提供**定性正确**的回转和 Z 形行为。
  定量匹配 SIMMAN 2020 基准在 ±5× 范围内（缺少 cross-flow drag 项）。

### 3.2 MPC 后端

| Backend | 类型 | 梯度 | 动力学模型 | 默认用于 |
|---------|------|------|-----------|----------|
| `casadi` (IPOPT) | 非线性优化 | 自动微分 | **可微 surrogate** (3-DOF 符号化) | B4-B7, Proposed |
| `scipy` (SLSQP) | 非线性优化 | 有限差分 | **完整 MMG callback** | 自动降级 |
| `sampling` | 随机探索 | 无 | **完整 MMG callback** | B3 仅 |

> **重要**: CasADi backend 使用可微 surrogate 动力学（提供 IPOPT 精确梯度），
> 而非完整 MMG 模型。完整 MMG 用于：(a) SLSQP/sampling backend 的 rollout；
> (b) CasADi 求解后的 post-solve 约束验证。论文必须明确区分两者。

### 3.3 安全距离三明治结构

```
d_safe_total = d_safe_eff          ← Layer 1: DynamicShipDomain (标量加性, 9项)
             + rho_tube             ← Layer 2: compute_tube_radius (8项保守求和)
             + κ·√(λ_max(Σ_rel))   ← Layer 3: 机会约束裕度 (χ² 2-DOF, 高斯假设)
```

### 3.4 动态船舶域

**实际实现**: 标量加性安全距离（非全椭圆/四元数/quaternion 域）。
每目标船输出单一各向同性标量半径 [m]。

### 3.5 物理效应路径

| 效应 | → 动力学 | → 管半径 | → 安全裕度 | → 代价惩罚 | → metadata |
|------|---------|---------|-----------|-----------|-----------|
| 浅水 | ✅ (阻力) | ✅ | — | ✅ | ✅ |
| 岸壁 | ✅ (吸力) | ✅ | ✅ | ✅ | ✅ |
| 船间 | ✅ (干扰力) | ✅ | — | ✅ | ✅ |
| 风 | ✅ (外力) | ✅ | — | — | ✅ |
| 流 | ✅ (运动学) | ✅ | — | — | ✅ |

### 3.6 目标船行为预测

7 种假设轨迹（trajectory-level worst-branch selection）:
1. 恒定速度
2. AIS 历史估计转向率
3. **COLREGs-role-aware** 机动（使用 `target_role`，非 `ownship_role`）
4. 航道约束
5. 非合规（向本船错误方向转向）
6. 紧急规避
7. 随机游走

> 这些是**保守行为包络**，不声称预测目标船真实意图。

## 8. Statistical Analysis

### 8.1 实验设计

- 每场景: ≥100 扰动 episodes
- 每 (episode, method): ≥5 seeds
- 每 (scenario, method): ≥500 独立仿真
- 种子: MD5 确定性哈希 (`hashlib.md5`), 跨 Python 版本可复现

### 8.2 统计检验

| 方法 | 用途 |
|------|------|
| Paired t-test | 配对场景-方法比较（正态性假设下） |
| Wilcoxon signed-rank | 非参数备选（小样本、非正态） |
| Cohen's d | 标准化均值差异效应量 |
| Cliff's delta | 非参数效应量 |
| Holm-Bonferroni | FWER 控制（族系错误率） |
| Benjamini-Hochberg | FDR 控制（假发现率） |
| Cluster bootstrap | 按真实 AIS episode ID 聚类重采样 |
| Mixed-effects model | Metric ~ Method + Scenario + (1|Seed) |

### 8.3 NaN/Inf 处理

- NaN 传递为 `None`（JSON null），**永不静默转 0**
- 配对检验仅用完整配对（报告 `n_excluded`）
- 汇总统计报告 `valid_count` 和 `nan_count`

## 10. Limitations

### 10.1 动力学模型

- 简化 MMG 提供**定性正确**行为，非定量 CFD/RANS 保真度
- 标准 MMG (KVLCC2/KCS) 产生物理上合理的转向行为，但定量 SIMMAN 匹配
  在 ±5× 范围内（缺少 cross-flow drag 和 hull-rudder interaction 项）
- 仅控制本船；目标船遵循预定义轨迹

### 10.2 感知与通信

- 完美本船状态知识（定位噪声通过 `own_position_std` 可配置进入）
- VHF/ASM 通信延迟未建模
- AIS 不确定性建模为对角高斯协方差（非高斯误差由 tube 半径部分覆盖）

### 10.3 MPC 后端

- **CasADi backend 使用可微 surrogate 动力学**，非完整 MMG
  - 完整 MMG 用于: (a) SLSQP/sampling 后端的 rollout; (b) CasADi 的 post-solve 验证
  - Surrogate 参数可通过 config 调整
- IPOPT 设置 5s wall-clock 超时；超时或 infeasible 时自动降级到 SLSQP

### 10.4 船舶域

- **标量加性模型**（各向同性），非全椭圆/四元数/quaternion 域
- 不得声称"方向相关域"或"完整 Fujii/Goodwin 域"

### 10.5 物理效应

- 保守有界扰动模型（基于发表系数），非 CFD 精度
- 浅水/岸壁/船间效应未经独立水池/实船试验验证
- 无冰区或低能见度建模

### 10.6 理论与形式化

- 管半径 = 8 项之和 → **保守设计**（假设所有扰动同时取最坏值并对齐）
- 机会约束基于高斯假设（χ² 2-DOF）
- CBF 和 fallback 增加保守性层级
- 递归可行性形式化分析（`analysis/stability.py`）当前为线性化框架

### ✅ 可以声称

- A reproducible, multi-source rule-aware, physics-enhanced, chance-constrained
  tube-MPC framework for ship collision avoidance in restricted waterways
- Empirical demonstration of safety-efficiency tradeoff improvement over
